## Estructura Recomendada de Lectura

### Para Usuarios Nuevos:
1. Comienza con el `README.md` principal (en la raíz del proyecto)
2. Lee `EJERCICIOS_PRUEBA.md` para entender las capacidades
3. Consulta `TEST_README.md` para probar la calculadora

### Para Desarrolladores:
1. `METODO_DOS_FASES_ANALISIS_IMPLEMENTACION.md` - Entender la lógica del solver
2. `CHECKPOINTS.md` - Historia del desarrollo
3. `DESPLIEGUE.md` - Configuración de producción

### Para Mantenimiento:
1. `TEST_README.md` - Ejecutar pruebas de regresión
2. `CHECKLIST_RENDER.md` - Validar despliegue
3. `requirements.md` - Verificar dependencias
